O que é Angular?

Aqui você verá os principais elementos que compõem este Freamwork e como utilizá-los.

História e funcionalidade

O framework é um conceito de Misko Hevery e Adam Abrons, engenheiros da Google. A empresa lançou a primeira versão do AngularJS em 2012 e tem mantido o serviço ativo desde então com atualizações e melhorias.

O AngularJS é um framework de código aberto usado para construir aplicativos para web baseados em uma única página dinâmica.

O AngularJS usa a arquitetura Model-View-Controller (MCV), que é usado no desenvolvimento de apps para a internet. Esse tipo de arquitetura consiste em:

  • Model (Modelo) – a estrutura de dados que gerencia a informação e recebe comandos do controle.
  • View (Visão) – a representação da informação.
  • Controller (Controle) – responde aos comandos e interage com o modelo.
